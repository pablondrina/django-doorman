# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-20

### Added
- Phone-first authentication with MagicCode (OTP)
- BridgeToken for chat-to-web session transfer
- IdentityLink model for User-Customer mapping
- WhatsApp Cloud API sender (optional)
- Email and console code senders
- Django admin interface
- Configurable templates and settings
